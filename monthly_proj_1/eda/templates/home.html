분석 목적

소비자 페르소나 유형 분석을 통해 개인화된 상품 구매 추천 시스템 설계

데이터 설명 및 출처

  • 설명
    • 고객 성격 분석은 기업의 이상적인 고객에 대한 상세한 분석입니다.
    • 이를 통해 기업은 고객을 더 잘 이해하고 다양한 유형의 고객의 특정 요구 사항, 행동 및 우려 사항에 따라 제품을 더 쉽게 수정할 수 있습니다.
    • 고객 성격 분석은 기업이 다양한 유형의 고객 세그먼트의 대상 고객을 기반으로 제품을 수정하는 데 도움이 됩니다.
    • 예를 들어, 회사 데이터베이스의 모든 고객에게 신제품을 마케팅하기 위해 비용을 지출하는 대신
    • 회사는 어떤 고객 세그먼트가 제품을 구매할 가능성이 가장 높은지 분석한 다음 해당 특정 세그먼트에만 제품을 마케팅할 수 있습니다.
  • 출처
    • Kaggle Competition - Customer Personality Analysis

    가설 4

    고객 가구의 자녀 수나 청소년 수가 높을 수록 불만 사항을 제기한 경우가 비례할 것이다
    {% for src in hypo4 %}
    {% if src == 'theory_4_1.png' %}
  • 자녀 수가 적을 수록 컴플레인 건수는 적으나, 자녀 수 표본의 불균형으로 인해 불만 제기 빈도 자체는 큰 의미가 없음을 알 수 있다
    • {% endif %} {% if src == 'theory_4_2.png' %}
  • 자녀 수에 따른 불만 제기 비율을 살펴봤을 때, 어린 자녀의 수가 많을 수록 불만 제기하는 비율이 더 놓은 것으로 나타나나, 청소년 자녀의 수와 불만 제기 비율은 비례하지 않는 것으로 나타난다
  • 이에 자녀 수와 무관하게 단순 자녀 유무에 따른 불만 제기 비율을 살펴볼 필요성이 존재한다
    • {% endif %} {% if src == 'theory_4_3.png' %}
  • 어린 자녀가 있는 가정의 경우, 불만 제기 비율이 어린 자녀가 없는 가정에 비해 약 2배로 눈에 띄게 높은 것을 알 수 있으나, 청소년 자녀가 있는 가정의 경우, 유의미한 차이가 존재하지 않는 것으로 나타난다
  • 하지만, 이는 청소년 자녀 수가 3이상인 표본이 존재하지 않아 엄밀한 증명은 불가한 것으로 보인다
    • {% endif %}
    {% endfor %}

    결론

    • 가설 1
      • 자녀 수와 식습관 지출은 반비례한다
      • 반대로, 자녀 수와 금 소비는 비례한다
    • 가설 2
      • 결혼 상태와는 관계없이 비슷한 지출을 하며, 와인 > 고기 > 금의 순서로 지출의 크기가 줄어든다
      • 결혼한 사람 > 동거 > 독신 > 이혼 순으로 구매 빈도가 높다
    • 가설 3
      • 소득이 0일 때를 제외하고, 소득이 적을 수록 금의 구매량이 감소한다
      • 소득순위가 높아질 수록 더욱 다양한 금액 투자가 이루어져 평균적인 금 구매량은 높을 것으로 추정된다
    • 가설 4
      • 자녀 수 혹은 자녀 존재 여부와 컴플레인 비율은 정비례한다
      • 아래의 그래프를 통해 청소년 자녀 존재 여부와 컴플레인 비율은 관계가 없음을 확인할 수 있다
      • 그러나 청소년 자녀 수가 3 이상인 표본이 없어 3 이상일 때는 컴플레인 비율이 증가할지는 증명이 불가하다
